Biểu mẫu trong html
        Sử dụng thẻ form
        Thẻ form:
        +Name Tên của form
        +Method-Phương thức Gửi dữ liệu của form(post, get)
        +action=""
     Tạo bảng trong html
        +dùng thẻ table
        +Hàng Tr - Cột Td
        +Thẻ bảng có header là thead
        +thẻ thân của bảng là tbody
     Thẻ input radio
        +Muốn chỉ có thể chọn 1 trong các input radio thì phải cho name giống nhau
        +Muốn chọn trc 1 input radio nào đó thì phải cho thuộc tính checked
     Thẻ Select
        +Thẻ select chứa các option
        +Thẻ option có thuộc tính quan trọng là value
        +Muốn chọn trước 1 option nào thêm thuộc tính selected
        +Muốn chọn nhiều option thêm thuộc tính multiple
     Thẻ textarea (cho phép nhập nhiều dòng văn bản)
        +Thẻ textarea có thẻ đóng và thẻ mở
     Thẻ submit
        +khi click chuột vào thẻ này tất cả dữ liệu trong form sẽ được
        submit đến đích đến được chỉ định trong thuộc tính action của thẻ form
 
     Thuộc tính style trong html
        +Là thuộc tính dùng để trang trí cho thẻ html
        +Trong thuốc tính style của html sẽ chứa các thuộc tính của css
        +2 thuộc tính đầu tiên của css: color(màu chữ),background-color(màu nền)
        +Thuộc tính css đc phân tách với giá trị của nó bằng dấu :
        +Khi có nhiều thuộc tính css phân tách với nhau bằng dấu ;
        +Thuộc tính font-size chỉ ra cỡ chữ cho phần văn bản
